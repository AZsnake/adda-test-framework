# IQ 参考波形

本目录包含两类 IQ 波形文件，布局相同：int16 交错 `I0, Q0, I1, Q1, ...`；`.bin` 小端序，`.WAVEFORM` 大端序。

## DAC 正弦参考（FPGA 板载播放）

AD9117 DAC 路径（`dac_wave_player` → `tx_iq_dsp` 2× 半带插值 → `tx_ddr_out`）的循环复数正弦播放文件。
使用 `tools/scripts/dac/wave_upload.py` 上传；用 `tools/scripts/dac/gen_ref_waveforms.py` 重新生成本目录全部正弦文件。

### 命名规范

```
sine_iq_<音调>_<电平>.<扩展名>
```

| 字段 | 含义 |
|------|------|
| `sine_iq` | 复数正弦，I/Q 交错 |
| `<音调>` | 音调频率，RKM 记法 — `9M00` = 9.00 MHz，`3M84` = 3.84 MHz |
| `<电平>` | 相对 s14 满幅（±8191）的幅度 dBFS：`-6dBFS`、`-12dBFS`、`-18dBFS` |
| `<扩展名>` | `.bin` = 小端序，`.WAVEFORM` = 大端序（内容相同，字节序相反） |

每个正弦文件固定参数（未写入文件名）：**Fs = 61.44 MSa/s/ch**（sys_clk/2），
**1 MB** = 262144 个 IQ 样点，**相干记录**（整数周期 → 无缝循环与相干 FFT）。

### 幅度约定（统一 14-bit）

样本为 **有符号 14 位（±8191 = 满幅）**，存储在 int16 中。TX 链 **1:1** 映射到 DAC，不做 `>>2` / ÷4。dBFS 与峰值码字对应关系（每 6 dB 减半）：

| 电平 | 峰值（s14） |
|------|------------|
| −6 dBFS  | 4096 |
| −12 dBFS | 2048 |
| −18 dBFS | 1024 |

−6 dBFS 为默认工作电平：为半带插值器通带过冲留有余量（峰值建议 ≲7200 ≈ −1 dB，避免 `tx_iq_dsp.reduce14` 饱和）。

### 当前正弦文件

| 文件 | 音调 | 电平 |
|------|------|------|
| `sine_iq_3M84_-6dBFS`  | 3.84 MHz | −6 dBFS  |
| `sine_iq_9M00_-6dBFS`  | 9.00 MHz | −6 dBFS  |
| `sine_iq_9M00_-12dBFS` | 9.00 MHz | −12 dBFS |

9 MHz 组构成 6 dB 步进幅度阶梯，可用于 SFDR-幅度扫描。每个文件同时提供 `.bin` 和 `.WAVEFORM` 两种格式。

## VSG 方波时钟（外部信号发生器）

供 **外部 VSG** 差分 IQ 端口使用的 20 MHz 方波时钟，**与 FPGA RTL 无关**。

| 文件 | 音调 | Fs | 说明 |
|------|------|-----|------|
| `sq_iq_20M00_125Msps` | 20.00 MHz | 125 MSa/s | I = Q 同相方波，满幅 ±8191 |

### 命名规范

```
sq_iq_<音调>_<Fs>.<扩展名>
```

| 字段 | 含义 |
|------|------|
| `sq_iq` | 方波时钟，I/Q 同相 |
| `<音调>` | RKM 记法 — `20M00` = 20.00 MHz |
| `<Fs>` | 采样率 — `125Msps` = 125 MSa/s |

相干循环单元：**25 IQ 样点 = 4 个完整 20 MHz 周期**（占空比约 52%，Fs/f 非整数比下的量化结果）。文件长度 1,048,500 字节（262125 IQ 对，10485 个周期）。

### 信号发生器配置

文件内样本以 0 为中心（±8191 满幅）；**共模与差分幅度在仪器端设置**：

| 参数 | 设置值 |
|------|--------|
| Sample rate | 125 MSa/s，循环播放 |
| I / Q | 差分输出，两路同相（文件已保证 `I == Q`） |
| Common mode | 0.6 V（每脚） |
| 差分幅度 | 0.6 Vpp（单脚 0.3 Vpp，摆幅 0.45 V ~ 0.75 V） |
| 单端峰值（若仪器按 peak 标定） | 0.15 V |

### 重新生成

```bash
python tools/vsg/gen_sq_iq_clock.py \
  --fs-hz 125000000 --tone-hz 20000000 \
  -o docs/wave/sq_iq_20M00_125Msps.bin \
  --waveform-out docs/wave/sq_iq_20M00_125Msps.WAVEFORM
```
